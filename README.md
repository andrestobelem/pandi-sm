# pandi-sm

A Smalltalk implementation written in Node.js — a Smalltalk language runtime
(lexer, parser, object model, and evaluator) running on the Node platform.

> **Status:** early development. There's no public API or CLI yet; this README
> will grow as the runtime takes shape.

## Repository layout

- `doc/research/` — background research and design notes. See
  [`doc/research/README.md`](doc/research/README.md) for the document format.
- `CLAUDE.md` / `AGENTS.md` — engineering conventions and agent instructions.
